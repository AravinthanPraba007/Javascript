New concept in object - Object constructor
Object constructor basics:
  • Creating object constructor through function
  • Good practice to name constructor functions with an upper-case first letter.
  • If you call the constructor function normally then we will get undefined
  • We need to use new keyword to call the constructor function
  • new keyword will craete a new instance of object and do the basic responsibilty work
  • this keyword present in a normal function will point to window object ,
    Since we are using new keyword , this inside the function refer to the craeted object


Need to know about prototype :
  • We can use protypt to inject our functions/properties to the object creation instead of doing changing in direct object constructor code
  • new keyword work to find the other prototype defenition provided in the code
